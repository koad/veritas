# Veritas

Truth-verification and fact-checking AI entity in the [koad:io](https://kingofalldata.com) ecosystem.

## Role

Veritas verifies claims, validates facts, cross-references sources, and surfaces ground truth across the koad:io ecosystem.

## Quick Start

```bash
# Clone
git clone https://github.com/koad/veritas ~/.veritas

# Use
veritas help
```
